投資人常見問題

VC、創投、銀行 ESG 分析師、企業投資部關於 AI 品牌可見度量化的 10 個常見方向。本頁僅作高層次說明,不揭露具體財務預測。

AI 品牌可見度如何量化?

Symcio 以 ABVI（AI Brand Visibility Index）作為觀察指標,範圍 0–100,由 Mention Rate、Average Rank、Sentiment Index 等公開可重現因子加權。ABVI 為基於公開 AI 引擎查詢結果的觀察性指標,可作為品牌在 AI 答案層的曝光參考,不構成投資建議或估值意見書。

Symcio 相較於 SimilarWeb、SEMrush 等工具的差異?

SimilarWeb 量化網站流量,SEMrush 量化 Google 排名,兩者皆為傳統網路時代的觀察工具。Symcio 觀察的是 ChatGPT、Claude、Gemini、Perplexity 四引擎回答中的品牌出現頻率與排名脈絡,屬於不同資料層,並不取代既有工具。Symcio 完整 ABVI 計算方法論於 GitHub 公開,可重現、可查核。

Symcio 的 API 接入方式?

企業合作方案提供 REST API 接入,可取得 ABVI 分數、歷史趨勢、競品 benchmark 等資料。費率與接入規格依使用情境客製,合格機構可申請技術 PoC。詳細條件請來信 sall@symcio.tw 索取資料。

BrandOS 的目標市場?

主要 TA 為:(1) 採購流程涉及 AI 搜尋的 B2B SaaS 與企業服務公司;(2) 有國際永續揭露要求的上市櫃公司;(3) 進行品牌投資與贊助成效量化需求的企業行銷部門。詳細市場規模分析、滲透假設與五年商業模型,於合格投資人 data room 提供。

主要競品有哪些?

類比參考:(1) 國際 AI 監測工具如 Profound、AthenaHQ、BrandRank.AI（以英文市場、單引擎為主）;(2) 傳統 SEO/品牌監測工具如 SEMrush、Brandwatch（加入 AI 功能）;(3) ESG 資料平台如 MSCI、Sustainalytics（覆蓋 ESG 評級但不含 AI 曝光）。Symcio 切入點為跨四引擎 + 繁中亞太語料 + ESG 資料對齊三軸,各家定位與功能組合不同,實際採購建議請依組織需求評估。

AI 引擎資料取得的合規性?

Symcio 透過 OpenAI、Anthropic、Google、Perplexity 等業者的官方 API 付費取用,遵守各家 Terms of Service 中「研究與分析」用途相關條款,不進行未授權爬取或逆向工程。測試 prompt 為公開方法論。客戶識別資料以 hash 匿名處理。歐盟客戶資料儲存於 EU 區域,流程依 GDPR 設計。

如何用 AI 品牌可見度觀察投資標的的品牌健康度?

觀察路徑:(1) 建立標的 ABVI 基準線（免費掃描即可）;(2) 對標同產業可比公司 ABVI 分佈;(3) 觀察 12 個月 ABVI 趨勢與標的營收、行銷投放的時間關聯。此為觀察性分析,不取代正式品牌估值或盡職調查,亦不構成證券投資建議。

Symcio 的營收模式?

產品分三層:單次報告（Audit,公開定價於 /pricing）、企業合作（自訂方案,Contact pricing）、未來訂閱方案（法人實體成立後上架）。詳細收入模型、毛利結構、客戶分布與成長預測,於合格投資人 data room 以 NDA 形式提供。公開頁面不揭露具體財務預測。

盡職調查可驗證哪些技術細節?

可公開驗證:(1) ABVI methodology 開源於 github.com/SALL911/BrandOS-Infrastructure,clone 後可獨立重現分數計算;(2) Schema.org / Wikidata 工具開源;(3) 公開 prompt 集與評分權重。NDA 後可提供:資料表 schema、實際客戶 ABVI 時序、引擎 API 帳單實況、產業覆蓋深度等。

Symcio 與金融資料終端整合的長期路徑?

長期路徑圖包含將 ABVI 資料以 REST API、Webhook、CSV 等格式提供給企業內部研究平台、CRM 或第三方資料終端,以及合作對接的可能。當前階段優先提供 REST API 與 Webhook,具體整合方式依客戶既有資料架構協議。任何提及 Bloomberg、SimilarWeb、SEMrush 等名稱僅作為市場類比,Symcio 不主張任何授權、合作、代表或背書關係。
